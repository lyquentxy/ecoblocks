# Embedded

Low-memory and legacy runtime route.

The modern EcoBlocks app targets Android API 21+ and uses Flutter. Android 4.4/API 19 is not part of the main APK.

If an API 19 product route is revived, keep it separate:

- Native Android API19 UI.
- Simple status, settings, logs, and saved-rule execution.
- No promise of full Blockly editing.
- Shared product principles, but not shared Flutter runtime.
