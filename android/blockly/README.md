# Blockly 积木块引擎

## 定位

Blockly 在 EcoBlocks 中是**可视化表示层**。核心价值在于：

1. 降低用户门槛——不需要写代码
2. 规则可视化——一眼看懂自动化逻辑
3. 教育场景——学生可以"拼积木"来理解编程

## 实现方式

基于 Google Blockly 开源库，WebView 容器加载。

## 魔改方向

- 自定义生态积木块（温度触发、光照控制、时间调度）
- 中文语言包
- 生态主题 UI 皮肤
- 规则 JSON 导出格式适配 EcoBlocks 协议

## 积木块分类

| 类别 | 示例 |
|---|---|
| 传感器 | 温度、湿度、水位、流量、光照 |
| 执行器 | 灯光、水泵、风扇、加热棒 |
| 时间 | 定时、日出日落、延时 |
| 逻辑 | 如果/那么/否则、比较、与或非 |
| 通知 | 推送、记录日志 |
